### Install dependencies:

```bash
npm install 

#or
yarn
```

### Run the development server:

```bash
npm run dev

#or
yarn dev
```

### Build for production AND generate static website into "out" folder:

```bash
npm run build

#or
yarn build
```

### Run in production:

```bash
npm start

#or 
yarn start
```
